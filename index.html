About This Project

These simulations are built entirely in vanilla HTML, CSS, and JavaScript — no frameworks, no build steps. Each page is self-contained and runs entirely in the browser. The project applies evolutionary game theory to the classic Rock-Paper-Scissors game to reveal surprisingly rich emergent behaviour.
